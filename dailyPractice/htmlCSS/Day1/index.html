切版任務 Day1 - HTML 標籤元素、CSS 判斷

依照下列 HTML 與 CSS 程式碼內容判斷，請分享以下 HTML 標籤

a. p、span、div、a 各為什麼顏色

Tips: 觀察哪個 class 最後宣告
( orange -> blue -> red -> yellow -> green )

  • p: green
  • span: green
  • div: red
  • a: green

b. p、span、div、a、img 是區塊元素（block）還是行內元素 （inline）

  • p: 區塊元素 (block)
  • span: 行內元素（inline）
  • div: 區塊元素 (block)
  • a: 行內元素（inline）
  • img: 行內元素（inline）
