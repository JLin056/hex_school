content-box

此為 box-sizing 屬性的預設值，width(寬)、height(高) 只包括內容本身的寬高，不包括 border、padding。
舉例：
當有一個 box 設定 width: 30px; height: 20px; border: 10px solid black;， 則此 box 總寬高：寬度: 50px, 高度: 40px

寬：300 + 16*2 + 3*2 = 338px

高：24*3 + 16*2 + 3*2 = 110px

border-box

width(寬)、height(高) 除了內容本身的寬高，還包括 border、padding
舉例：
當有一個 box 設定 width: 30px; height: 20px; border: 10px solid black; box-sizing: border-box;， 則此 box 總寬高：寬度: 30px, 高度: 20px

寬：300px

高：24*3 + 16*2 + 3*2 = 110px
